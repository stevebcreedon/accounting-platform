# Phase 4 — UI Design Contract

> Visual and interaction contract for site structure and pages. Generated by gsd-ui-researcher, verified by gsd-ui-checker.
>
> Phase 4 builds every page beyond the article template: homepage, guides hub, category archives, find-accountant placeholder, and static pages (about, contact, privacy, terms, 404). All design tokens, typography, color, and spacing are inherited from Phase 1. Card patterns and component conventions established in Phase 3 are reused directly. This phase introduces several new layout patterns (hero sections, card grids, filter pill strips, static prose pages) but no new design tokens.

| Property | Value |

### Accent Usage in Phase 4

The burnt-orange accent (#E8720C) is used exclusively for:

1. **Hero CTA link** -- "Browse all guides" text link in burnt-orange-500, hover to burnt-orange-600
2. **Active filter pill** -- burnt-orange-500 background with white text
3. **Category pill on article cards** -- burnt-orange-50 bg with burnt-orange-600 text (inherited from Phase 3)
4. **"Comprehensive Guide" badge** -- burnt-orange-500 bg with white text (inherited from Phase 3)
5. **Hyperlinks** in prose content on static pages (inherited from Phase 1)
6. **Focus-visible outlines** on all interactive elements (inherited from Phase 1)
7. **Find-accountant CTA button** -- burnt-orange-500 bg, white text, burnt-orange-600 hover

Accent is NOT used for: section headings, category card borders, inactive filter pills, 404 page elements, footer text.
