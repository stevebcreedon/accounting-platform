2. Rewrite `src/app/globals.css` with Tailwind directives, base layer overrides, and article-content component class:

```css
@tailwind base;
@tailwind components;
@tailwind utilities;

@layer base {
  html {
    scroll-behavior: smooth;
  }

  ::selection {
    background-color: rgba(232, 114, 12, 0.15);
    color: #1C1917;
  }

  :focus-visible {
    outline: 2px solid #E8720C;
    outline-offset: 2px;
  }
}

@layer components {
  .article-content {
    @apply prose prose-lg max-w-article mx-auto;
    @apply prose-headings:font-heading;
    @apply prose-a:text-burnt-orange-500 prose-a:underline-offset-2;
    @apply prose-blockquote:border-burnt-orange-500;
  }
}
```

Base layer details:
- `scroll-behavior: smooth` on html for anchor link scrolling
- `::selection` uses burnt-orange at 15% opacity with charcoal text
- `:focus-visible` uses 2px solid burnt-orange (#E8720C) with 2px offset for accessibility
- `.article-content` component class combines prose plugin with brand overrides (Satoshi headings, burnt-orange links and blockquote borders)
